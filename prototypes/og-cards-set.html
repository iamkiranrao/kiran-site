What this is. Each OG card uses the existing bento hero illustration as the background and overlays a cream content card with eyebrow + title + byline. The image stays the same for every page within a content type (every teardown share uses og-teardowns.png; every blog post uses og-blog.png). The page-specific title comes from the og:title meta tag, which Slack/LinkedIn render below the image.

How to wire. In each page's <head>:
<meta property="og:image" content="https://kiranrao.ai/images/og/og-teardowns.png">
<meta property="og:title" content="How I'd've Built It: Instagram">
<meta property="og:description" content="...">

To regenerate. Edit copy or styling in scripts/generate-og-cards.py, then run python3 scripts/generate-og-cards.py. All 9 cards regenerate in a few seconds. Add a new card type by appending an entry to the OG_CARDS list.
